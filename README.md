# Nur Nachrichten - tagesschau.de RSS Feed

## Features

- Responsive web app for cross-platform compatibility
- Simple, minimal, elegant
- Five sections: Home, Inland, Ausland, Wirtschaft, Wissen
- Full title and description in feed, click out to tagesschau.de
- Performance and caching: localStorage to cache articles for faster loading, background updates when switching between feeds
